About Relier

Imagine a group of friends located at different parts of a city and are planning to meet at a restaurant. Now this situation becomes hassle as they are facing difficulties to decide a restaurant which is nearby to everyone. Normally they will spend a lot of time for finding an ideal restaurant by switching to many applications such as maps and restaurants applications.

Relier completely resolves this problem by suggesting a list of restaurants by entering their locations. Thus making the whole process fast and hassle free.

How it Works?

  • A User has to create an account through mobile number which will be verified via otp

    After registering, the user will be redirected to Discover tab.

    Discover tab will show our current location which is editable. We can search and add our friends through contact list. After adding we can edit their location manually. Once all our friends are added, a chat group will be automatically created .Now click on Search Restaurants .

  • After clicking the Search Restaurants , we will get the list of restaurants which we can go through swiping Each result will show how far is the restaurant from all the users and how much time will it take for each user to reach. A list of favourable restaurant can be created by selecting Add List option.

    We can share the list by clicking on Share with Friends.

    After sharing the list, we will be redirected to Alert tab displaying all the selected restaurants.

  • In chat group we can discuss with our friends and the user will choose the selected restaurant in Alert tab .Once the restaurant is selected, User need to add event name, date and time of the event.

    Now click on Save Event.

    After successful creation of an event, we can check the upcoming event in Events under Home tab

Can Relier provide the list of restaurants which is near for one particular person with respect to others? icon

Yes, Relier has a feature which can provide the list of restaurants near to one particular person alone. We have added a star feature which allows to filter the search result based on this scenario.

Relier can provide a list of restaurants ideal for maximum 10 people.
